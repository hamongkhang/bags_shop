== Description ==

<a href="https://crestaproject.com/cresta-facebook-messenger-demo-page/" taget="_blank">Plugin Demo</a> | <a href="https://crestaproject.com/downloads/cresta-facebook-messenger/" taget="_blank">Plugin Homepage</a>

With <strong>Cresta Social Messenger</strong> you can allow your users or customers to contact you via Facebook Messenger simply by clicking on a button. Users may contact you directly in private messages on your Facebook page and continue the conversation on Facebook Messenger.
You can use the floating button which is positioned at the bottom right of the site or a button inside post or pages using the <strong>[cresta-facebook-messenger text="Need Help?" icon="yes" position="top"]</strong> shortcode.
Cresta Social Messenger is compatible with WooCommerce and WPML.

<strong>Some features</strong>
<ul>
	<li>Add the button to open Facebook Messenger at bottom right of your site</li>
	<li>Use the shortcode <strong>[cresta-facebook-messenger]</strong> to add the Facebook Messenger button in your posts or pages</li>
	<li>Choose which pages to showo the Facebook Messenger button</li>
	<li>Mobile version: click the button and automatically open Facebook Messenger App (if installed)</li>
	<li>and more...</li>
</ul>

<strong>PRO version features</strong>
<ul>
	<li>Change position of the box button</li>

== Installation ==

1. Unzip cresta-facebook-messenger.zip
2. Upload the folder 'cresta-facebook-messenger' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress Dashboard
4. Go to your WordPress Dashboard under "Main Menu -> Settings-> Cresta Facebook Messenger" to set the options


== Frequently Asked Questions ==

= Where I find the options page? =
After installing the plugin, using the WordPress Admin menu, go to "Settings-> Cresta Facebook Messenger"

= Can I hide the Facebook Messenger box in a few pages? =
Yes, just edit the page/post you want to hide the box, find the metabox called "Cresta Facebook Messenger" and check the box "Hide Cresta Facebook Messenger in this page".

= Can I use my Facebook profile or group with this plugin? =
No, this plugin works only with Facebook pages (no profiles or groups)
